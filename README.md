# Commands:
Start Command 

    :/

End Command 

    ::/
    
File Write On RunTime

    ::/file/write/runtime

File Write 

    ::/file/write/name/line_num/content/line_num
    or
    ::/file/write/content/line_num/name/line_num
             
File Read 

    ::/file/read/name/line_num
    
Delete File On Runtime

    ::/file/delete
